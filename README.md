A simple battle royale script, you can choose the location and see who's the last one to survive inside the zone!

## Features

- PVP Zone 
- Time & Zone range full custom
- Damage if you're outside the Zone
- Admin command to start/end the Zone
- You can put everywhere, even outside cayo perico
- Admin whitelist for Damage
- Just drag and drop


## Installation

- Download the last build and drag it into your server
- Change the config and locales
- Add into your server.cfg

```cfg
  ensure spreaxBattleRoyale
```
